vagrant-dev-mongodb

Wie man vagrant nutzt kann man zum Beispiel hier nachlesen: https://github.com/michaelgruczel/vagrant-hello-world

einfach 'vagrant up' aufrufen und schon läuft die box und mongo horcht auf 27018.
